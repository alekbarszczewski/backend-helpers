# backend-helpers

Some very opinionated helpers to set up graphql backend quickly and keep your code DRY.

## Install

```sh
$ yarn add backend-helpers
```

## Usage

```js
import { loadStore, loadGraphql, createGraphqlApp } from 'backend-helpers'
```
